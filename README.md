
# Broadcast Analyzer

Analyze and test livestream URLs using Flask.

## Features
- Login protected
- Analyze streaming link (token, protocol, ffprobe info)
- Record 10 seconds from stream

## Requirements
- Python 3.x
- ffmpeg, ffprobe installed and in PATH

## How to Run
```bash
pip install flask
python app.py
```
Open in browser: http://localhost:5000
